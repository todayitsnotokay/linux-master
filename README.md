# Не забыть изменить заголовок

### $ apropos
> Ищет переданный аргумент, в названиях и описаниях команд. Может помочь  в поиске команды по её функционалу.

*Пример*\
`apropos copy`

### $ type
> Отображает то, как интерпретируется комада в терминале. Может быть полезна для обработки элиасов.

*Пример*\
`type copy`
