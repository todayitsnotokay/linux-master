# Не забыть изменить заголовок

| Command | Description |
| --- | --- |
| $ apropos | Ищет переданный аргумент в названиях и описаниях команд. Может помочь в поиске команды по её описанию.<br>*Пример*: `apropos send` |
| $ type | Отображает то, как интерпретируется команда в терминале. Может быть полезна для обработки alias.<br>*Пример*: `type cp` |
| $ sed | Потоковый редактор текста.<br>*Пример*: `sed -i 's/unix/linux/' /home/text.txt` |
|  |  |
